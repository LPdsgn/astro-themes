# astro-themes

![Astro](https://img.shields.io/badge/astro-%232C2052.svg?style=for-the-badge&logo=astro&logoColor=white)
![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white)

Perfect dark mode in Astro with no flash. An Astro integration that mirrors the behavior of [next-themes](https://github.com/pacocoursey/next-themes).

To see how to get started, check out the [package README](./packages/astro-themes/README.md)

## Features

- [x] Perfect dark mode in 2 lines of code
- [x] System setting with `prefers-color-scheme`
- [x] No flash on load (both SSR and SSG)
- [x] Sync theme across tabs and windows
- [x] Force pages to specific themes
- [x] Class or data attribute selector
